# MM_Tool
## Make Model Tool

- Streamlit Library를 활용한 ML 모델 튜닝 Tool 제작
  
  1. 데이터셋 등록 및 (추세, 박스플롯) 시각화
  2. 전처리, Feature Selection
  3. ML 모델 선택 및 내부 파라미터 조정
  4. Anomaly Detect 선택 및 파라미터 조정
  5. 생성한 모델들 성능 비교
